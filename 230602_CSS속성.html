CSS3 속성 미니테스트

01 다음 빈칸에 알맞은 단어를 넣으시오.

  1. ( 박스 )속성에는 margin 속성, border 속성, padding 속성, width 속성, height 속 성이 포함됩니다.
  2. 테두리 두께는 ( border-width )속성으로 입력하고, 테두리 형태는 ( border-style ) 속성으로 입력 하며, 테두리 색상은 ( border-color )속성으로 입력합니다.
  3. ( border-radius )속성을 사용하면 박스의 테두리를 둥글게 만들 수 있습니다.
  4. ( display )속성에 none 키워드를 적용하면 태그가 화면에서 보이지 않습니다.
  5. background-( img )속성은 배경 이미지를 넣을 때 사용합니다.
  6. background-( size )속성은 배경 이미지 크기를 지정합니다.
  7. background-( repeat )속성은 배경 이미지의 반복 형태를 지정합니다.
  8. background-( attachment )속성은 배경 이미지를 화면에 고정합니다.
  9. background-( position )속성은 배경 이미지의 배치를 지정합니다.
  10. font-( size )속성은 글자 크기를 지정합니다.
  11. font-( family )속성은 글꼴을 지정합니다.
  12. ( serif )글꼴(명조체)은 모서리가 뾰족한 글자를 의미합니다.
  13. ( Sans-serif )글꼴(고딕체)은 모서리가 네모진 글자를 의미합니다.
  14. font-( style )속성은 글자 스타일(기울기 등)을 지정합니다.
  15. font-( width )속성은 글자 두께를 지정합니다.
  16. ( text-align )속성은 글자 정렬을 지정합니다.
  17. ( line-height )속성은 글자 높이를 지정합니다.
  18. ( position )속성은 요소의 위치 지정 형식을 설정할 때 사용합니다.
  19. ( overflow )속성은 내용이 요소 크기를 벗어나 모두 보여 주기 힘들 때 어떻게 보여 줄지 를 지정합니다.
  20. ( box-shadow )속성은 박스에 그림자를 부여합니다.
  21. ( text-shadow )속성은 글자에 그림자를 부여합니다.

02 다음 질문에 OX를 표시하시오.

  1. inline 형식의 태그에는 width 속성과 height 속성이 적용되지 않으며, margin 속성과 padding 속성은 좌우로만 적용됩니다. ( X )
  2. inline-block 형식의 태그는 width 속성과 height 속성이 지정되며, margin 속성과 padding 속성이 상하좌우로 적용됩니다. ( O )
  3. block 형식의 태그는 width 속성과 height 속성이 지정되며, margin 속성과 padding 속 성이 상하좌우로 적용됩니다. ( O )
  4. background-image 속성에는 이미지를 하나만 지정할 수 있습니다. ( X )
  5. font-family 속성에 글꼴을 여러 개 지정하는 것은 아무런 의미가 없습니다. ( X )
  6. 일반적으로 웹 브라우저의 글자 기본 크기는 16px입니다. ( O )
  7. 기본적인 글꼴 두께 값은 400입니다. ( O )
  8. inline 형식의 태그에는 text-align 속성을 지정해도 의미가 없습니다. ( X )
  9. 요소들은 기본적으로 절대 좌표계를 사용합니다. ( O )
  10. z-index 속성의 숫자가 클수록 앞에 위치합니다. ( O )
  11. overf low 속성을 hidden으로 지정하면 영역을 넘은 부분을 보이지 않게 만듭니다. 이때 스크롤을 사용해 보이지 않게 된 부분을 볼 수 있습니다. ( X )
