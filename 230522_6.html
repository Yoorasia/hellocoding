CSS 문법 형식

CSS기초 1

1 선택자, 2 중괄호 안(속성, 값) 으로 이루어져 있습니다.

CSS 설명(주석)을 남기려면 /* 텍스트 */ 이렇게 표기합니다.

현재 사용중인 CSS 호출 방식은 inline방식, embed(internal-내부스타일시트)방식 linked(external-외부스타일시트)방식, 이렇게 3가지의 방식으로 호출하는데, 그 중 내부스타일시트 방식을 사용하고 있습니다.

  • 인라인 : <태그 style="css 코드">
  • 엠베드 : 헤드 태그 안에 <style>/*css 코드*/<style>
  • 링크드 : <link rel="stylesheet" href="css문서가 들어있는 파일 경로">
