웹 개요와 실습 환경 구축 미니테스트

01 다음 괄호 안을 채우시오.

( 인터넷 )은 전 세계를 연결하는 국제 정보 통신망으로, 컴퓨터나 스마트폰 같은 디지털 기기로 연결되어 사람들이 정보를 공유할 수 있는 공간입니다.
( 하이퍼링크 )는 인터넷에서 문서 사이를 쉽게 이동할 수 있는 기능입니다.
( W3C )는 HTML 표준을 비롯한 웹 표준안을 제작하거나 제안하는 일을 하는 국제적인 웹 표준화 단체입니다.
( 웹 브라우저 )는 웹에 공개된 정보를 탐색할 수 있게 하는 프로그램입니다.
( 플러그인 )은 사용자 PC에 프로그램을 추가로 설치해 웹 브라우저 기능을 확장하는 방법입니다.
( 웹 2.0 )은 서로 다른 사용자가 함께 새로운 콘텐츠를 창조할 수 있는 시대를 의미합니다.
( WHATWG )는 Web Application 1.0 표준을 작성한 단체입니다. 현재는 HTML W/G라는 이름으로 변경했습니다.
Web Application 1.0 표준은 새로운 웹 표준으로 채택된 후 ( HTML5 )표준이라는 이름으로 변경되었습니다.
웹은( 요청 )과 ( 응답 )이라는 간단한 형태로 동작합니다.
( UR )은 웹에서 어떤 대상을 구분하는 주소를 의미합니다.
( URI )는 웹에서 어떤 대상을 구분하는 방법들을 총칭합니다.
요청하는 쪽을 ( 클라이언트 )(사용자)라고 하며, 응답하는 쪽을 ( 서버 )(제공자)라고 합니다.
서버 프로그램은 사용자와 먼 곳에 있기 때문에( 벡엔드 )프로그램이라고 합니다.
클라이언트 프로그램은 사용자와 가까운 곳에 있기 때문에 ( 프론트엔드 ) 프로그램이라고 합니다.
( 마크업 )은 웹 페이지의 서식이나 구조를 표현하는 정보입니다.
( CSS )는 HTML 페이지에 스타일을 지정하는 스타일시트를 작성할 때 사용하는 언어입니다.
( ECMAScript )는 자바스크립트의 표준 명칭입니다.
( 자바스크립트 )는 HTML 페이지에서 사용자 반응 등을 처리하는 스크립트를 작성하는 언어입니다.
( 일렉트론 )은 HTML5 기반의 데스크톱 애플리케이션 개발 엔진입니다.
( 리액트 네이티브 )는 페이스북에서 만든 모바일 애플리케이션을 개발할 수 있게 하는 엔진입니다.
( 크로스 플랫폼 개발 )은 한 번 개발해서 여러 장치(플랫폼)에 대응하는 것을 의미합니다.
